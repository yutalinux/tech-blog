# Tech Blog

## ディレクトリ構成

| ディレクトリ           | 概要                         |
| ---------------------- | ---------------------------- |
| `app`                  | メインのフロントエンド       |
| `contents`             | 記事データ                   |
| `packages`             |                              |
| ├ `config`             | ツール類のコンフィグファイル |
| ├ `parser`             | マークダウンのパーサー       |
| ├ `remark-note-plugin` | remark プラグイン            |
| ├ `styles`             | ブログのスタイル (css)       |
| ├ `ui`                 | ui コンポーネント (react)    |
